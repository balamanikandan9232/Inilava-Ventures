# 🌐 Inilava India Ventures - Landing Page

A sleek and responsive landing page for **Inilava India Ventures**, crafted with HTML, CSS, and JavaScript. This project includes a functional **Call to Action** button that allows users to request a quote via **EmailJS**, making it easy to capture leads and inquiries.

## 🚀 Features

- Responsive design for desktop and mobile
- Clean layout with modern UI elements
- Call-to-action button integrated with EmailJS
- Smooth scroll and interactive sections
- Modular and readable codebase

## 🛠️ Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Styling with Flexbox and media queries
- **JavaScript** – DOM manipulation and EmailJS integration
- **EmailJS** – Client-side email service for quote requests

## 📩 Quote Request Integration

The "Get a Quote" button triggers an EmailJS function that sends user input (name, email, service, message) directly to your configured email address. No backend required.

🧠 How to Use
Clone the repository 

Open index.html in your browser

Configure your EmailJS credentials in script.js

Deploy using GitHub Pages or any static hosting platform
